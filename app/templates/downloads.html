Below you can access CFM geospatial data for fault traces in both 2D and 3D formats.

GeoPackage Files (.gpkg)

The GeoPackage files available here are designed for use with GIS software, including ArcGIS. These files offer efficient, and versatile storage for both 2D and 3D fault trace data:

  • Download CRESCENT Cascadia 3D Crustal Fault Surfaces (GeoPackage)

Metadata Table (.csv)

This comma-separated values (CSV) file provides a summary metadata table for faults included in the CRESCENT Community Fault Model (CFM). Each row corresponds to an individual fault or fault section and includes key identification, geometric, kinematic, and provenance information such as fault name, region and jurisdiction, dip, rake, slip sense, depth extent, slip-rate estimates used in the National Seismic Hazard Model (NSHM), and references. The table is intended as a lightweight index to the CFM dataset, enabling quick inspection, filtering, and integration with GIS, modeling workflows, and web applications, while detailed fault geometries and 3D model representations are provided in separate data files. Explanatory notes in selected fields document data lineage, modeling assumptions, and known limitations.
